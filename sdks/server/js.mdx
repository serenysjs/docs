  "The Serenys server-side JS SDK allows you to interact with all Serenys endpoints on
  the server side."
---

Currently, the SDK is compatible with [Node.js](https://nodejs.org/en/).

## Installing

```bash
# NPM
npm i serenys

# Or with Yarn
yarn add serenys

# Or with Pnpm
pnpm add serenys

# Or with Bun
bun add serenys
```

## Creating an SDK Instance

```js JS
import { Serenys } from "serenys";

// Export the Serenys SDK instance so you can use it throughout your codebase
export const serenys = new Serenys();

// Example: Getting an Discord User
await serenys.discord.users.get("299262860440371202");

// Example: Getting an invite from a Discord server
await serenys.discord.invites.get("discloud");

// Example: Getting an Instagram user
await serenys.instagram.users.get("hhh.e_c.v");

// Example: Getting a post from an Instagram user
await serenys.instagram.posts.get("Ctt93euSakq");

// Example: Getting the last 4 posts of an Instagram user
await serenys.instagram.feed.get("hhh.e_c.v");
```
