---
title: "Get User"
api: "GET https://serenys.xyz/api/discord/users/299262860440371202"
description: "This endpoint allows you to get information from a Discord account"
---

### Response

<Expandable title="Parameters">
	<ResponseField name="user" type="object">
		Basic user information
	</ResponseField>
  
  <ResponseField name="user_profile" type="object">
    User profile information, such as your badges, up to your about me
	</ResponseField>

  <ResponseField name="boost" type="object">
	  User boost information
  </ResponseField>

  <ResponseField name="legacy_username" type="string">
	  The old username
  </ResponseField>

  <ResponseField name="connected_accounts" type="array">
    Accounts connected to your Discord profile
	</ResponseField>
</Expandable>
