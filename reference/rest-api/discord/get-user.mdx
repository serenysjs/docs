---
title: "Obter informações de um usuário"
api: "GET https://serenys.xyz/api/discord/users/299262860440371202"
description: "Este endpoint te permite obter a informação de uma conta do Discord"
---

### Resposta

<Expandable title="Parâmetros">
	<ResponseField name="user" type="object">
		Informações básicas do usuário
	</ResponseField>
  
  <ResponseField name="user_profile" type="object">
    Informações do perfil do usuário, como suas insígnias, até seu sobre-mim
	</ResponseField>

  <ResponseField name="boost" type="object">
	  Informações sobre o boost do usuário
  </ResponseField>

  <ResponseField name="legacy_username" type="string">
	  O antigo nome de usuário
  </ResponseField>

  <ResponseField name="connected_accounts" type="array">
    Contas conectadas ao seu perfil do Discord
	</ResponseField>
</Expandable>
