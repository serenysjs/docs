<Note>
	Normalmente, você não precisa usar a API REST diretamente. Nós temos suporte a
	[SDKs](/sdks/overview) para usar a Serenys com várias linguagens e estruturas.
	O principal motivo pelo qual você precisará usar a API REST diretamente é se
	estiver escrevendo uma API wrapper da Serenys em uma linguagem para a qual
	ainda não temos suporte.
</Note>

#### URL Base da API

```
https://serenys.xyz/api/
```
